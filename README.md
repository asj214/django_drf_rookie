# django_drf_rookie


### environments
- python 3.9
- django 3.2.3

### commands
```sh
# migrate and fixtures
sh init.sh

# runserver
python manage.py runserver_plus

# show url list
python manage.py show_urls

# shell
python manage.py shell_plus
```

### seeders
```sh
# user
python manage.py users

# posts
python manage.py posts

# commands
python manage.py comments
```
